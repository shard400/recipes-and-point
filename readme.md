# Рецепты и точка — Веб-приложение для поиска рецептов

Данный проект представляет собой веб-приложение «Рецепты и точка» — агрегатор кулинарных рецептов с интеграцией внешнего API Edamam, системой авторизации пользователей, личным кабинетом и административной панелью.

## Требования к системе
- **Веб-сервер:** Apache (например, в составе XAMPP или OpenServer)
- **PHP:** версия 7.4 или выше
- **СУБД:** MySQL (версия 5.7 или выше)
- **Браузер:** Любой современный браузер (Chrome, Edge, Firefox, Safari)

## Инструкция по запуску проекта

Для успешного запуска и работы веб-приложения необходимо выполнить следующие действия:

### 1. Подготовка серверного окружения
1. Установите локальный веб-сервер (рекомендуется **XAMPP** или **OpenServer**).
2. Запустите модули **Apache** и **MySQL**.

### 2. Размещение файлов проекта
1. Скопируйте все файлы проекта (`.php`, `.css`, папку `img/`, `Vector.png` и др.) в корневую директорию локального сервера.
   - Для XAMPP: `C:\xampp\htdocs\pract\`
   - Для OpenServer: `C:\OSPanel\domains\pract\`

### 3. Настройка базы данных
1. Откройте **phpMyAdmin** (обычно по адресу `http://localhost/phpmyadmin`).
2. Создайте новую базу данных с именем **`recipes_and_point`**.
3. Выполните SQL-скрипт из файла `database.sql` (если он прилагается) для создания таблиц, либо вручную выполните SQL-запросы для создания таблиц:
   - `users` (пользователи)
   - `categories` (категории)
   - `chefs` (шеф-повара)
   - `recipes` (рецепты)
   - `collections` (коллекции)
   - `recipes_collections` (связь рецептов и коллекций)
   - `favorites` (избранное)
   - `blog_posts` (статьи блога)
   - `ingredients` (ингредиенты)
   - `recipe_ingredients` (связь рецептов и ингредиентов)
4. При необходимости заполните таблицы начальными данными.

### 4. Настройка подключения к базе данных
1. Откройте файл **`db.php`**.
2. Убедитесь, что параметры подключения соответствуют вашей локальной настройке:
   - `$host = 'localhost';`
   - `$user = 'root';`
   - `$pass = '';` (оставьте пустым, если пароль не установлен)
   - `$dbname = 'recipes_and_point';`

### 5. Запуск веб-приложения
1. Убедитесь, что сервер Apache и MySQL запущены.
2. Откройте любой веб-браузер.
3. В адресную строку введите: **`http://localhost/pract/`**.
4. Откроется главная страница приложения «Рецепты и точка».

### 6. Доступ к административной панели
1. На странице входа (`http://localhost/pract/auth.php`) авторизуйтесь с учетными данными администратора (по умолчанию: `admin@recipes.ru` / `123456`).
2. После входа в верхнем меню появится ссылка **«Админ-панель»**.
3. В административной панели доступны: просмотр и редактирование всех таблиц, а также просмотр представлений (Views).

### 7. Работа с поиском рецептов (API Edamam)
1. Для корректной работы поиска рецептов в файле `search.php` должны быть указаны актуальные ключи API Edamam.
2. По умолчанию используются ключи, указанные в коде. При необходимости их можно заменить на собственные, зарегистрировавшись на сайте [developer.edamam.com](https://developer.edamam.com/).
